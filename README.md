Farm Connect is a console-based farmer-to-market supply chain management system built using Java.

## Features
- Farmer registration and product management (add/update/delete products)
- Buyer product search by name or farmer
- Cart system and order placement
- Order tracking for both farmers and buyers
- Exception handling for invalid inputs and out-of-stock products

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Exception Handling
- Eclipse IDE

## Project Modules
- Farmer Module
- Buyer Module
- Product Management
- Order History

This project simulates a digital marketplace where farmers and buyers interact efficiently.
